# Union

**Meet Together. Build Together.**  
A video meeting app built with WebRTC. No downloads, no accounts, 1-click to join.

Live: https://joinunion.vercel.app  
Repo: https://github.com/solomonenyinnaya14-a11y/Union-

### Why Union
Zoom makes you download. Google Meet makes you sign in.  
Union: Click "Start a Call" → Share link → You’re in. 3 seconds.

Built for teams and friends.

### Features
- **Instant Rooms**: Unique link for every meeting
- **No Login Required**: Join from any browser
- **HD Video & Audio**: Peer-to-peer with WebRTC
- **Screen Sharing**: Present without installing anything  
- **Lightweight**: Loads fast on mobile and desktop
- **Free**: Always free for 1:1 and small groups

### Tech Stack
| Layer | Tech |
| --- | --- |
| **Frontend** | HTML5, CSS3, Vanilla JavaScript |
| **Backend** | Node.js, Express.js |
| **Realtime** | Socket.io for signaling |
| **Video** | WebRTC |
| **Hosting** | Vercel + GitHub CI/CD |
